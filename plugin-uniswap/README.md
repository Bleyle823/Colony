# @elizaos/plugin-uniswap

A plugin for interacting with Uniswap V4 on supported chains.

## Features
- Get quotes for token swaps (Single-hop & Multi-hop)
- Execute swaps using Universal Router
- Support for RWA assets like xStocks/mF-ONE

## Configuration
Requires `UNISWAP_RPC_URL` and `EVM_PRIVATE_KEY` in your `.env`.
